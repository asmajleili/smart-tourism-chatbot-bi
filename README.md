# 🧭 Smart Tourism Chatbot + BI Dashboard


## 📌 Description

Ce projet consiste à développer un assistant touristique intelligent
basé sur un Large Language Model (LLM).

Le système permet :

- Recommandation de destinations
- Suggestions d'hôtels et restaurants
- Personnalisation selon le profil utilisateur
- Analyse BI des interactions
